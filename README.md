# ATM Interface

## Description
A Java console-based ATM Interface that allows users to check balance, deposit money, withdraw money, and exit the application. The project demonstrates Object-Oriented Programming concepts using Java.

## Features
- Check account balance
- Deposit money
- Withdraw money
- Input validation
- Menu-driven interface
- Object-Oriented Programming (OOP)

## Technologies Used
- Java
- Scanner Class
- Classes and Objects
- Methods
- Switch Case
- Loops

## How to Run
1. Compile the program:
   javac ATMInterface.java
2. Run the program:
   java ATMInterface
